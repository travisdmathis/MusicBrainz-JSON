# MusicBrainz-JSON

A Ruby Wrapper for the MusicBrainz JSON API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mbjson'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mbjson
